# Orleans.GpuBridge.Backends.ILGPU

ILGPU backend implementation for Orleans GPU Bridge.

## Features
- ILGPU GPU kernel execution
- Cross-platform GPU support
- Device discovery and management
- Memory allocation and transfers

## Usage
This package provides ILGPU backend support for Orleans.GpuBridge framework.
